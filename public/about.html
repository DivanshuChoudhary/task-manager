🚀 Dockerized Task Manager

Dockerized Task Manager is a full-stack task management application developed using modern web technologies.

Users can create, update, delete and complete tasks with a clean, responsive and interactive interface.

✨ Features

  • ✅ Add Tasks
  • ✅ Edit Tasks
  • ✅ Delete Tasks
  • ✅ Mark as Completed
  • ✅ Search Tasks
  • ✅ Dashboard Analytics
  • ✅ Responsive Design
  • ✅ Docker Support
  • ✅ Express REST API

🛠 Tech Stack

  • HTML5
  • CSS3
  • JavaScript (ES6)
  • Node.js
  • Express.js
  • Docker
  • Git & GitHub
